# Relatorio em LaTeX para Prism

Este diretorio e um projeto LaTeX pronto para abrir no Prism.

Arquivos principais:

- `main.tex`: relatorio em formato ABNT usando `abntex2`.
- `referencias.bib`: referencias usadas no texto.
- `latexmkrc`: configuracao para compilar com `latexmk`.

No Prism, importe ou abra esta pasta e use `main.tex` como arquivo principal. O Prism e um workspace nativo em LaTeX, entao o projeto foi deixado em fonte LaTeX, sem PDF gerado no repositorio.

Antes de entregar, preencha no inicio de `main.tex`:

- nome completo em `\autor`;
- cidade em `\local`;
- instituicao e disciplina em `\instituicao`.
